# Plan Feature Workflow

Execute the **Planning Phase** of the Agentic SDLC.

## Steps

1. **Read the Planning Agent instructions**
   Read `.agent/skills/01-planning-agent/SKILL.md` to understand the planning process.

2. **Get the feature request**
   Ask the user for a high-level feature description if not already provided.

3. **Execute PLAN_001 — Create Epic**
   Create `docs/planning/EPIC-{FEATURE}.md` with:
   - Epic title, business context, goals, scope, success criteria

4. **Execute PLAN_002 — Create User Stories**
   Break the Epic into detailed User Stories with:
   - As a / I want to / So that format
   - Acceptance criteria for each story
   - Priority and size estimates

5. **Verify**
   - Confirm `docs/planning/EPIC-{FEATURE}.md` exists
   - Confirm it contains at least 3 user stories
   - Confirm each story has acceptance criteria
