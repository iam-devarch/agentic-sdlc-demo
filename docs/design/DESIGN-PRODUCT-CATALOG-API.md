# API Design: Product Catalog

> Generated by the **Design Agent** (`02-design-agent`) using skill DESIGN_001.

## Overview

REST API for managing products in the catalog. Supports full CRUD operations with pagination, validation, and structured error responses.

## Base Path

```
/api/v1/products
```

---

## Endpoints

### GET /api/v1/products — List Products

| Parameter | In    | Type    | Required | Default | Description            |
|-----------|-------|---------|----------|---------|------------------------|
| page      | query | integer | No       | 0       | Page number (0-based)  |
| size      | query | integer | No       | 20      | Page size (1-100)      |

**Response 200 OK:**
```json
[
  {
    "id": 1,
    "name": "Widget Pro",
    "description": "A premium widget",
    "price": 29.99,
    "createdAt": "2025-01-01T00:00:00Z",
    "updatedAt": "2025-01-01T00:00:00Z"
  }
]
```

---

### GET /api/v1/products/{id} — Get Product

| Parameter | In   | Type | Required | Description |
|-----------|------|------|----------|-------------|
| id        | path | long | Yes      | Product ID  |

**Response 200 OK:** Single product object  
**Response 404 Not Found:** Error response

---

### POST /api/v1/products — Create Product

**Request Body:**
```json
{
  "name": "Widget Pro",
  "description": "A premium widget",
  "price": 29.99
}
```

| Field       | Type    | Required | Constraints              |
|-------------|---------|----------|--------------------------|
| name        | string  | Yes      | 1-255 characters         |
| description | string  | No       | Max 1000 characters      |
| price       | decimal | Yes      | Greater than 0           |

**Response 201 Created:** Created product with generated `id` and timestamps  
**Response 400 Bad Request:** Validation error response

---

### PUT /api/v1/products/{id} — Update Product

Same request body as creation. Returns updated product.

**Response 200 OK:** Updated product  
**Response 404 Not Found:** Error if ID doesn't exist  
**Response 400 Bad Request:** Validation error

---

### DELETE /api/v1/products/{id} — Delete Product

**Response 204 No Content:** Success  
**Response 404 Not Found:** Error if ID doesn't exist

## Data Models

### Product (Domain)
| Field       | Type          | Description                  |
|-------------|---------------|------------------------------|
| id          | Long          | Auto-generated identifier    |
| name        | String        | Product name                 |
| description | String        | Product description          |
| price       | BigDecimal    | Product price                |
| createdAt   | LocalDateTime | Creation timestamp           |
| updatedAt   | LocalDateTime | Last update timestamp        |

### ProductDto (Response)
Same fields as Product domain model.

### CreateProductRequest (Request)
| Field       | Type       | Validation                        |
|-------------|------------|-----------------------------------|
| name        | String     | `@NotBlank`, `@Size(max = 255)`   |
| description | String     | `@Size(max = 1000)`               |
| price       | BigDecimal | `@NotNull`, `@Positive`           |
